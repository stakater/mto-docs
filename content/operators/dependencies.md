# Multi Tenant Operator Dependencies

The Multi Tenant Operator is built upon and integrates with the following two operators:

- **Template Operator**
- **Hibernation Operator**

For detailed documentation on each, please refer to their respective documentation sites:

- [Template Operator Documentation](https://docs.stakater.com/template-operator/latest/)
- [Hibernation Operator Documentation](https://docs.stakater.com/hibernation-operator/latest/)
